# Web Technologies Website

This project is created for the Web Technologies course assignment.

## Project Description
This website is designed using HTML and CSS.  
It displays course lectures and provides separate pages for each lecture.

## Features
- Homepage with lecture list
- Separate pages for each lecture
- Simple CSS styling
- Navigation links

## Technologies Used
- HTML
- CSS
